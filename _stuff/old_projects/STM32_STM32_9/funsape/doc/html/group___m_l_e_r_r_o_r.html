Detailed Description

Motion Library - Error definitions. Definition of the error codes used within the MPL and returned to the user. Every function tries to return a meaningful error code basing on the occuring error condition. The error code is numeric.

  • (44) INV_ERROR_DIVIDE_BY_ZERO
